0. Overview — The Five Forms & the Terminology We Use

Two labels for each form Genki English 文法用語 (Japanese term)

Genki introduces these one lesson at a time. This page gathers all five so the shared machinery (the -a-, -e-, られる patterns) is visible at once. Both names are given because we use both in class.

Genki term (English)文法用語 (our Japanese term)Core meaningGenki L
Volitional form意向形いこうけい / 意志形いしけい"let's / shall I / I intend to" — 〜（よ）うL15
Potential form可能形かのうけい"can / be able to" — 〜（ら）れる・〜えるL13
Passive form受身うけみ / 受身形うけみけい"be —ed (by …)" — 〜（ら）れるL21

Causative-passive使役受身しえきうけみ"be made to do (unwillingly)" — 〜（さ）せられるL23
Two terms that recur everywhere
  • U-verb = 五段ごだん動詞どうし (Genki "u-verb", Group 1). Conjugates by shifting the final -u vowel.
  • Ru-verb = 一段いちだん動詞どうし (Genki "ru-verb", Group 2). Conjugates by dropping る.
  • Irreg. = 不規則ふきそく動詞どうし — only する and くる.

1. 意向形いこうけい Volitional Form — 〜（よ）う "let's / I think I'll …"

Formation (plain volitional)
  • U-verb (五段ごだん): final -u → -o + う  (のむ → のもう)
  • Ru-verb (一段いちだん): drop る + よう  (たべる → たべよう)
  • Irregular: する → しよう　／　くる → こよう
  • Polite: verb ます-stem + ましょう  (のみましょう)
GroupDictionaryVolitional 〜（よ）うNote
U-verb飲のむ nom-u飲のもう nom-ouむ→も+う
U-verb行いく ik-u行いこう ik-ouく→こ+う
U-verb待まつ mats-u待まとう mat-ouつ→と+う
Ru-verb食たべる tabe-ru食たべよう tabe-youる→よう
Irregularする suruしよう shiyou—
Irregularくる kuruこよう koyou—
Basic usage
  1. Suggestion "let's": いっしょに 昼ひるごはんを 食べよう。Let's eat lunch together.
  2. Offer "shall I?": 荷物にもつを 持もちましょうか。Shall I carry your bag?
  3. Intention 〜（よ）うと思おもう: 週末しゅうまつは 勉強べんきょうしようと思おもっています。I'm thinking of studying this weekend.
  4. Attempt 〜（よ）うとする: 寝ねようとしたとき、電話でんわが 鳴なった。Just as I was about to sleep, the phone rang.

2. 可能形かのうけい Potential Form — "can / be able to"

Formation
  • U-verb (五段ごだん): final -u → -e + る  (のむ → のめる)
  • Ru-verb (一段いちだん): drop る + られる  (たべる → たべられる)
  • Irregular: する → できる　／　くる → こられる

「ら」抜ぬき言葉ことば: in casual speech ru-verbs often drop ら → たべれる, みれる. Recognize it, but write たべられる in formal work.

GroupDictionaryPotentialNote
U-verb飲のむ nom-u飲のめる nom-eruむ→め+る
U-verb話はなす hanas-u話はなせる hanas-eruす→せ+る
U-verb行いく ik-u行いける ik-eruく→け+る
Ru-verb食たべる tabe-ru食たべられる tabe-rareruる→られる
Irregularする suruできる dekiruspecial
Irregularくる kuruこられる ko-rareru—
Basic usage — note the particle shift を → が
  1. 日本語にほんごが 話はなせます。I can speak Japanese. (object を usually becomes が)
  2. 辛からいものが 食たべられません。I can't eat spicy food.
  3. Ability vs. possibility: ここから 富士山ふじさんが 見みられる。You can see Mt. Fuji from here.
  4. The potential is itself a ru-verb: 飲のめます／飲めない／飲めて.
Note — 見える / 聞こえる

見みえる (be visible) and 聞きこえる (be audible) describe spontaneous perception, not learned ability — separate from the 可能形 above.

3. 受身うけみ Passive Form — 〜（ら）れる "be —ed (by …)"

Formation
  • U-verb (五段ごだん): final -u → -a + れる  (う-ending → われる)
  • Ru-verb (一段いちだん): drop る + られる
  • Irregular: する → される　／　くる → こられる

Careful: for ru-verbs the passive and potential look identical (たべられる). Context tells them apart.

GroupDictionaryPassive 〜（ら）れるNote
U-verb読よむ yom-u読よまれる yom-areruむ→ま+れる
U-verb言いう i-u言いわれる i-wareruう→われる
U-verb取とる tor-u取とられる tor-areruる→ら+れる
Ru-verb見みる mi-ru見みられる mi-rareruる→られる
Irregularする suruされる sareru—
Irregularくる kuruこられる ko-rareru—
Basic usage — frame: [Aは] [Bに] V-passive
  1. Direct passive: 私わたしは 先生せんせいに ほめられました。I was praised by the teacher.
  2. Thing as subject: この歌うたは 世界中せかいじゅうで 歌うたわれています。This song is sung all over the world.
  3. Suffering / adversity passive（迷惑めいわくの受身うけみ): 雨あめに 降ふられた。I got rained on (to my inconvenience).
  4. 弟おとうとに ケーキを 食たべられた。My little brother ate my cake (on me).
Note — "by" is に

The agent is marked with に (sometimes から for information sources). The adversity passive can even take an intransitive verb (降る), which has no English equivalent — hence the "suffering" nuance.

4. 使役しえき Causative Form — 〜（さ）せる "make / let someone do"

Formation
  • U-verb (五段ごだん): final -u → -a + せる  (う-ending → わせる)
  • Ru-verb (一段いちだん): drop る + させる
  • Irregular: する → させる　／　くる → こさせる
GroupDictionaryCausative 〜（さ）せるNote
U-verb飲のむ nom-u飲のませる nom-aseruむ→ま+せる
U-verb待まつ mats-u待またせる mat-aseruつ→た+せる
U-verb買かう ka-u買かわせる ka-waseruう→わせる
Ru-verb食たべる tabe-ru食たべさせる tabe-saseruる→させる
Irregularする suruさせる saseru—
Irregularくる kuruこさせる ko-saseru—
Basic usage — [Causer]は [Causee]に／を V-causative
  1. を = make (coercion), intransitive: 先生せんせいは 学生がくせいを 立たたせた。The teacher made the students stand.
  2. に = let (permission): 母ははは 子こどもに 遊あそばせた。Mom let the child play.
  3. Transitive → causee takes に: 母ははは 子こどもに 野菜やさいを 食たべさせた。Mom made/let the child eat vegetables.
  4. Ask to be allowed: 少すこし 考かんがえさせてください。Please let me think a little.
Note — one を per clause

A transitive verb's object already holds を, so the causee must be に (the 二重にじゅうヲ格かく制約せいやく / "double-を constraint"). See the fuller Causative & Benefactive page for 〜させてあげる／くれる.

5. 使役受身しえきうけみ Causative-Passive — 〜（さ）せられる "be made to do"

Formation = causative + passive

causative 〜（さ）せる + passive 〜られる → 〜（さ）せられる

Frame: [Sufferer]は [Instigator]に (… を) V-causative-passive — usually against one's will.

GroupCausativeCausative-passiveContraction
U-verb飲のませる飲のませられる→ 飲まされる
U-verb待またせる待またせられる→ 待たされる
U-verb (す)話はなさせる話はなさせられる(no される: さ+さ avoided)
Ru-verb食たべさせる食たべさせられる—
Irregularさせる／こさせるさせられる／こさせられる—
Basic usage
  1. 私わたしは 部長ぶちょうに お酒さけを 飲のまされた。I was made to drink (sake) by my boss.
  2. 弟おとうとは 母ははに 部屋へやを 掃除そうじさせられた。My brother was made to clean his room.
  3. 毎日まいにち 残業ざんぎょうさせられて、疲つかれた。I'm made to work overtime every day and I'm exhausted.
The whole system at a glance
  • 〜（さ）せる = I make/let someone. 〜（さ）せられる = I am made to.
  • The contraction 〜される (飲まされた) only applies to u-verbs, and not to す-ending u-verbs (話させられる stays long).

6. Express Practice — 練習れんしゅう

A. Volitional（意向形 〜（よ）う）
  1. 行いく → ________　　2. 食たべる → ________　　3. する → ________　　4. くる → ________
B. Potential（可能形）— then rewrite with が
  1. 話はなす → ________　　2. 食たべる → ________　　3. する → ________
  2. Rewrite: 私わたしは かんじを 読よむ。→ 私わたしは かんじ（　）________。can read kanji
C. Passive（受身）
  1. ほめる → ________　　2. 言いう → ________　　3. する → ________
  2. Make passive: 先生せんせいが 私わたしを しかった。→ 私わたしは 先生せんせい（　）________。I was scolded
D. Causative & Causative-passive（使役・使役受身）
  1. Causative: 待まつ → ________　　食たべる → ________
  2. Choose に／を: 先生せんせいは 学生がくせい（　）立たたせた。made them stand
  3. Causative-passive: 私わたしは 先輩せんぱいに 1時間じかん（待まつ）________。use the contracted form
