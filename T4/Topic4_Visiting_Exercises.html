ウォームアップ / WARM-UP

話はなしてみましょう

  1. 知しりあいの家いえに行いったことがありますか。Have you visited someone's home?
  2. 日本にほんの家いえのげんかんで、何なにをしますか。What do you do in a Japanese entryway?
  3. 訪問ほうもんしたとき、どんなことをよく話はなしますか。What do you talk about when visiting?

ペアで話はなしましょう。 (Talk with a partner, ~2 min)

B. 家族かぞくのことば （じゅんび②・p.55）

正ただしいことばを選えらびましょう。家族かぞくについて話はなすとき、どんなことばを使つかいますか。

  1. 私わたしの（a 父ちち ・ b お父とうさん）はエンジニアです。
  2. うちは、（a 妻つま ・ b おくさん）と（a 子こども ・ b お子こさん）2人りの、4人家族にんかぞくです。

2. きく / LISTENING

B. 外国生活がいこくせいかつの思おもい出で （3-①・p.58）

山本やまもとさんの家いえにカールさんがあそびに来きました。(1) どんなことを話はなしていますか。 (2) （　）に入はいる文ぶんを選えらびましょう。

a アメリカに行いったこと、ありますか　b アメリカでの生活せいかつはどうでしたか　c いつごろですか　d 私わたしもそのころニューヨークで勉強べんきょうしていたんですよ

カール：山本やまもとさんは、（① a アメリカに行ったこと、ありますか）。

ようこ：カールさん、私わたしたち、ニューヨークに住すんでたんですよ。

カール：え、ほんとに？（②　　　　　　）。

ようこ：2004年ねんから2010年ねんまで、約やく6年間ねんかんです。

カール：へえ、（③　　　　　　）。

ようこ：ぐうぜんですね。

カール：（④　　　　　　）。

ようこ：ことばがわからなくてこまったけど、親切しんせつな人ひとが多おおくてたすかりました。

かずお：そうだね。あと、家族かぞくでよく旅行りょこうしたね、ようこ。おぼえてるか。グランドキャニオン、すごかったよね。

ようこ：ええ、あれはすばらしかった。わすれられないけしきだよね。

かずお：ああ。ぼくも、人生観じんせいかんが変かわった。

カール：へえ、そうですか。

3. ぶんけい / GRAMMAR

A. 家族かぞくのよび方かた （2-②・p.56）

家族かぞくをよぶとき

  • 「しょう、お姉ねえちゃんは？」
  • 「あ、お父とうさん、こっちよ。」

紹介しょうかいするとき

  • 「カールさん、むすこのしょうです。」
  • 「むすめのめぐみです。」
  • 「うちのしゅじんです。」

3. ぶんけい / GRAMMAR

B. ていねいたい・ふつうたい 1 （3-②・p.58）

つぎの文ぶんはだれに話はなしていますか。（　）に書かきましょう。

  1. 私わたしたち、ニューヨークに住すんでたんですよ。　ようこ →（カール）
  2. ぐうぜんですね。　ようこ →（　　　　）
  3. 親切しんせつな人ひとが多おおくてたすかりました。　ようこ →（　　　　）
  4. 家族かぞくでよく旅行りょこうしたね。　かずお →（　　　　）
  5. おぼえてるか。　かずお →（　　　　）
  6. わすれられないけしきだよね。　ようこ →（　　　　）

友ともだちに話はなすときと、家族かぞくに話はなすときは、どんなちがいがありますか。

4. はなす / SPEAKING

ロールプレイ ＆ 発表はっぴょう

A. 家族かぞくを紹介しょうかいする

「＿＿さん、紹介しょうかいします。うちの母ははです。今いま、英語えいごをならってるんですよ。」

「はじめまして。いつも むすめ が おせわになって(い)ます。」

「こちらこそ、いつも ＿＿さん に おせわになって(い)ます。」

B. 外国生活がいこくせいかつの思おもい出で

  • いつ、どこに住すんでいましたか。
  • どんな経験けいけんや思おもい出でがありますか。
  • 今いま、どう思おもいますか。（〜て、なつかしいです）

文ぶんを2つ以上いじょうならべて、少すこし長ながく話はなしましょう。（♪070）

ふりかえり / CAN-DO CHECK

できるようになりましたか？

  • お客きゃくを家いえの中なかにあんないすることができる。
  • 家族かぞくをお客きゃくに紹介しょうかいすることができる。
  • 外国がいこくなどで生活せいかつした経験けいけんや思おもい出でについて話はなせる。
