8.1: (人が) NをV(他動詞) / NがV(自動詞)

Grammatical Construction:
  • Transitive (他動詞): (人が) NをV (action denoted; an agent acts on N).
  • Intransitive (自動詞): NがV (change-of-state denoted; N changes by itself).
  • Japanese encodes the Trs/Intr distinction phonetically in verb pairs.
Example

Verb pairs in the textbook:

  • 電気/エアコン: つけます(他, 2G) / つきます(自, 1G) ; けします(他, 1G) / きえます(自, 2G)
  • まど/ドア: あけます(他, 2G) / あきます(自, 1G) ; しめます(他, 2G) / しまります(自, 1G)
  • おゆ/水: だします(他, 1G) / でます(自, 2G) ; とめます(他, 2G) / とまります(自, 1G)

Sentences:

  1. ホテルの人が電気をつけます。/ 電気がつきます。
  2. ベッドのそばの電気が きえないので、受付に電話しました。
  3. ホテルの人は、へやのエアコンを つけました。
  4. へやに入ったら、まどが あいたままでした。
  5. ホテルの人は、トイレの水を 止めました。
Note:
  1. Activity (他動詞) + ています = action in progress: 「電気をつけています」(turning on).
  2. Change-of-state (自動詞) + ています = achieved state: 「電気がついています」(is on).
  3. Many other pairs: こわします/こわれます, よごします/よごれます, わかします/わきます, etc.

8.2: V1.stem+ながら V2: doing V1 while doing V2

Grammatical Construction:
  • [... V1.ます+ながら] V2: V1 and V2 are done at the same time by the same subject.
  • V2 is the main action; V1 is the simultaneous accompanying action.
Example
  1. ホテルの人は、歩きながら、あべさんにホテルの歴史を 説明しました。
  2. パウロさんは、カフェでコーヒーを 飲みながら、メールをチェックしました。
  3. 私は、マッサージを してもらいながら、ねむってしまいました。
  4. 子どもたちは、おかしを 食べながら、ゲームをしていました。
  5. アリさんは、サインを 見ながら、出発ゲートをさがしました。
  6. 私のむすめは、空港で 泣きながら、友だちとわかれました。
  7. お酒を 飲みながら、ゆっくり映画を見ます。
  8. コンピューターで、ずっと 仕事をしながら、(すごします)。
  9. ガイドブックを 読みながら、旅行の計画をたてます。
  10. 外国語の勉強を しながら、話しながら、(すごします)。
  11. となりに座った人と 話しながら、すごします。
Note:
  1. The subject of V1 and V2 must be the same person: BAD ボブがピアノを弾きながら、トムは歌いました。
  2. 音楽を ききながら ご飯を食べます (V2 = main: eating; V1 = simultaneous: listening).
  3. Do not use to express ``while X happens'' between two different events; use 〜あいだ(に) instead.
