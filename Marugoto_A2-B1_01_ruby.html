1.1: イA/ナA+さ: Forming Nouns from Adjectives (名詞化めいしか)

Grammatical Construction:
  • イA: すばらしい+さ → すばらしさ
  • ナA: かんたんな+さ → かんたんさ
Example
  1. 長友ながとも選手せんしゅのプレーのすばらしさにかんどうしました。
  2. ルールのかんたんさが、サッカー人気にんきのりゆうの１つです。
Note:
  1. +さ turns an adjective into an abstract noun (the -ness of adj).
  2. Compare with Pred_stem+すぎる (Genki L12.2): same stem-based morphology, different suffix.

1.2: V.ます: Forming Nouns from Verbs (名詞化めいしか)

Grammatical Construction:
  • V: さそいます → さそい (an invitation)

The V.stem (= V.ます minus ます) can stand alone as a noun.

Example
  1. 勉強べんきょうが忙いそがしいから、友ともだちのさそいをことわりました。
Note:
  1. Cf. Genki L7.4 V.stem+に いく/くる, L11.1 V.stem+たい, L12.2 _stem+すぎる — same V.stem behavior.
  2. Not every verb has a productive stem-noun; common ones: さそい, 終おわり, はじまり, 考かんがえ, etc.

1.3: と/で/へ/から/まで+の: Particle+の as a Noun Modifier

Grammatical Construction:

[NP+Particle]+の+NOUN
A PP (particle phrase) can modify a following NOUN by attaching の.

Example
  1. 来月らいげつの マリナーズとの試合しあい、いっしょに行いきましょう。 (と+の)
  2. JFサッカー場じょうでの試合しあいは、年ねんに15回かいぐらいです。 (で+の)
  3. JFサッカー場じょうへの行いき方かたをサイトでしらべました。 (へ+の)
  4. ファンからのプレゼントは、Tシャツでした。 (から+の)
  5. 駅えきからの道みちは、20分ふんぐらいです。 (から+の)
  6. ファンへのメッセージは、サイトで読よめます。 (へ+の)
Note:
  1. を/が/には［ふつう］のでつなげません: *友ともだちがの本ほん; *ご飯はんをの時間じかん
  2. Compare with Genki L10.3 ADJ/NOUN+の (the white one, Bob's): same +の nominalizer/linker.

1.4: V.なければなりません / V.なきゃいけません

Grammatical Construction:
  1. たべない+ければなりません / いけません — must eat
  2. おいしくない+ければなりません / いけません — must be delicious
  3. 学生がくせいじゃない+ければなりません / いけません — must be a student
Example
  1. 土曜日どようびに父ちちの知しりあいをむかえに行いかなければなりません。
  2. 土曜日どようびに父ちちの知しりあいをむかえに行いかなきゃいけないんです。
  3. カーラさんはあさってまでに、レポートを 出ださなければなりません。
  4. ホセさんは明日あした、大使館たいしかんにしょるいをとりに行いかなければなりません。
  5. あさって、パクさんは出張しゅっちょうしなければなりません。
Note:
  1. 〜なければなりません is phonetically reduced as 〜なきゃいけません in spoken Japanese.
  2. Use 〜なきゃいけないんです (〜んです) when declining an invitation because you must do something else.
  3. Past form: 〜ない+ければなりませんでした / いけませんでした.
  4. Compare with Genki L6.4 V.て+はいけません (must not).

1.5: 命令形めいれいけい — Forming the Command Form (Affirmative)

Rule by verb group:
  • Ru-verb (一段いちだん): drop る, add ろ. たべる → たべろ
  • U-verb (五段ごだん): change the final -u sound to -e. のむ → のめ
  • Irreg. する → しろ　／　くる → こい

1.6: 禁止形きんしけい — Negative Command, & Usage

Negative command (uniform for all verbs):
  • Add な to the dictionary form (the plain present-affirmative form).
  • [V-辞書形じしょけい] + な = "Don't do V!" — のむ → のむな　／　たべる → たべるな　／　する → するな　／　くる → くるな
Example
  1. がんばれ！ — Go for it! / Hang in there! (cheering)
  2. はしれ！　まけるな！ — Run! Don't lose! (at a sports event)
  3. 早はやく ねろ。 — Go to sleep, now. (parent → child)
  4. ここに 車くるまを とめるな。 — Don't park here. (sign / order)
  5. 「待まて」と言いわれました。 — I was told to "wait." (quoted command)
Note:
  1. A very strong, authoritative command (senior→junior, parent→child, coach→player). It can also be an encouraging shout (がんばれ, まけるな).
  2. Blunt and masculine — avoid with strangers or superiors; use V-てください / V-ないでください in polite situations.
  3. You will meet it in cheers, road signs (とまれ), emergencies, manga, and reported commands.
